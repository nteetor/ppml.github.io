Working with GitHub

Need to double check what that git command was?
{% terminal %} $ git --help > usage git: ... > ... {% endterminal %}
Check that your username and email are configured.
{% terminal %} $ git config --global user.name $ git config --global user.email {% endterminal %}

At this point you have added all your file changes or, in our analogy, written down all your updates in the letter.

When you commit changes it is as though you are stamping and sealing your letter's envelope. You have declared the letter complete (git add --all) and now there is only one additional step to sending the letter on its way

{% terminal %} $ git commit -m "[your message]" > {% endterminal %}
