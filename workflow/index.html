This is a guide to walk you through the tools we use within the PPML group. This guide is not intended to shackle you to our workflow, quite the opposite. The guide is intended to show and convince you of the benefit of the tools and systems we have chosen to use. Afterwards, we hope you will find yourself eager or excited to incorporate the new workflow.

A quick double-check

Before diving into the git installation, we can double-check if git is already installed.

To check if git is already installed on your computer follow the steps.

  1. Open up your computer's Terminal application.
  2. Once you have the terminal open, type and run the following command.

    git --version

  3. If git is installed you will see an output similar to the following.

    git --version

    git version 2.10.1 (Apple Git-78)

  4. If git is not installed you will see an output similar to the following.

    git --version

    bash: git: command not found

  5. At this point, if git is installed you only need to proceed if you would like to upgrade to the latest version of git. At the time of writing the latest version of git was 2.14.2.

  6. If git is not installed or you would like to upgrade your version of git click below to proceed to the download page.
