Resources

The resources listed below are used in PPML projects or useful when working on PPML projects. Clicking on a resource will take you to a corresponding web page.

cran mirror

 A GitHub organization which is a read-only mirror of CRAN. This GitHub organization, also named cran, has a repository for each package on CRAN. This is a helpful resource when package authors have not linked to or have not at all put their source code online.

Network Models, Workshop 1

 Harvard workshop materials on EpiModel.

lookup

 An R package for better exploration of R, C, and C++ source code.

statnet

 The home page for the statnet group.
